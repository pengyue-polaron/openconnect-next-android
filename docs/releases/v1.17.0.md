# OConnect v1.17.0

OpenConnect Next is now OConnect. This release completes the product rebrand and gives the app its own Android identity.

## Highlights

- Renames the app, interface, documentation, store metadata, themes, logs, and project identifiers to OConnect.
- Changes the Android application ID to `io.pengyue.oconnect`.
- Aligns remote API permissions, intent actions, FileProvider authorities, Quick Settings labels, and localized copy with the new identity.
- Preserves OpenConnect terminology where it refers to the supported protocol and native VPN engine.
- Renames branded icon source assets for future OConnect releases.

## Important upgrade note

Because the Android application ID changed from `io.pengyue.openconnectnext` to `io.pengyue.oconnect`, this release installs as a separate app. Existing profiles and settings are not automatically migrated from OpenConnect Next.
