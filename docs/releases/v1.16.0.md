# OpenConnect Next v1.16.0

This release gives OpenConnect Next a consistent dashboard-led experience across the complete VPN journey.

## Highlights

- Replaces the legacy top-tab home screen with one dashboard for disconnected, connecting, waiting-for-input, and connected states.
- Keeps the selected VPN profile, primary connection action, connection status, duration, and traffic information on the first screen.
- Moves logs into Connection details and hides unavailable network fields while disconnected.
- Unifies profiles, settings, FAQ, About, SecurID, dialogs, and empty states around the same Material 3 visual system.
- Widens authentication dialogs, improves repeated-field labels, and keeps password visibility and save-password controls easy to reach.
- Preserves advanced configuration depth while making common settings shorter and easier to scan.

## Verification

- Unit tests, Android lint, debug build, and release build pass.
- The debug APK installs and launches on the Android 36 emulator.
- Dashboard, profile management, add-profile dialog, authentication, connection details, nested logs, settings, FAQ, About, SecurID, and overflow actions were exercised.
- No fatal application exception was observed in the final runtime log check.
