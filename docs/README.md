# OConnect Documentation

OConnect is a focused Android client for OpenConnect-compatible SSL VPN
gateways. This directory contains the public product, packaging, and release
documentation that accompanies the source code.

## Product and Brand

- [Brand guide](brand.md) — product name, positioning, voice, color system,
  asset locations, and screenshot standards.
- [README](../README.md) — public project overview, installation, features,
  screenshots, and build instructions.

## Distribution

- [Fastlane metadata](../fastlane/metadata/android/en-US) — store title,
