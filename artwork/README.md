# OpenConnect Next artwork

The launcher icon uses a monochrome globe-and-shield mark on a charcoal background.

## Source artwork

- `openconnect-next-icon.svg` — complete launcher icon with background.
- `openconnect-next-icon-foreground-white.svg` — white mark on a transparent background.
- `openconnect-next-icon-foreground-black.svg` — dark mark on a transparent background.

All source files use a `108 × 108` view box and can be scaled without loss.

## Android resources

- `drawable-anydpi-v23/ic_launcher.xml` — vector fallback for Android 6 and 7.
- `drawable-anydpi-v26/ic_launcher.xml` — adaptive icon definition.
- `drawable-anydpi-v33/ic_launcher.xml` — adaptive icon with a monochrome layer for themed icons.
- `drawable/ic_launcher_foreground.xml` — adaptive foreground artwork.
- `drawable/ic_launcher_monochrome.xml` — single-color themed-icon artwork.
- `values/ic_launcher.xml` — launcher background color.

The vector resources replace the previous density-specific raster launcher icons.
